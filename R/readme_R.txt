README 
---------------------------------------------------------------------------------------

This folder contains R script to generate visualisations for the study:
Vaccine coverage and equity impact in Kenya

plot.R - plot script that generates
    (i) plot_socioeconomic_geographic_maternal_child_birthsetting_coverage.jpg (and .eps figures)
   (ii) plot_aor.jpg (and .eps figures)


kenya_map.R - plot script that generates
  (iii) Kenya_vaccination_coverage.jpg (and .eps figures)

---------------------------------------------------------------------------------------

data files 

    (i) data_coverage.csv
   (ii) data_aor.csv
  (iii) ken_coverage.csv
